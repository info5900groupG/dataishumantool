Data Analysis is Human Work

A tool that shows every decision we make as we analyze data


About

The aim of this project is to provide a better sociological understanding of data analytics, and to advance data-analytic research and training by improving the fit between technical tools and solutions to real world problems.

This tool is a proof-of-concept that uses the analysis of Reddit Global News headlines and DJIA stock prices as an example for how the human work of data analysis can be shown.

The research is based on the hypothesis that stock prices are affected by current events.
